# Code of Conduct

We aim to provide a welcoming, harassment-free experience for everyone contributing to Abyssforge.

## Our pledge

- Be respectful and inclusive.
- Use welcoming and inclusive language.
- Accept constructive criticism.
- Focus on what is best for the community and project.

## Our standards

- Demonstrate empathy toward other community members.
- Provide constructive feedback with supporting evidence.
- Refrain from harassment, trolling, or derogatory comments.

## Enforcement

Project maintainers are responsible for clarifying standards and enforcing them. Violations may result in temporary or permanent bans from project spaces.

## Scope

This Code of Conduct applies to all project spaces (GitHub, chat, events) and when individuals represent the project.

## Reporting

Report abusive behavior to `conduct@abyssforge.io` (placeholder). The core team will review and respond promptly while maintaining confidentiality.
